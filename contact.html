Contacts

If you encounter any problems using or downloading the software, please see the Gihub Issue page, or create a new issue on the ISSIE GitHub repository. Any feedback and suggestions are also welcome - we keep feature request issues and usually manage to implement them!

If you want to use ISSIE in an educational context contact Dr. Tom Clarke, Department of Electronic & Electrical Engineering, Imperial College London.
